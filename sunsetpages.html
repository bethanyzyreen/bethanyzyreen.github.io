About the Project

Sunset-Themed Design

Sunset Pages was designed around a warm, sunset-inspired color palette, blending the calm of golden hour with the cozy feeling of getting lost in a good book.

Curated Photo Galleries

Book listings are paired with curated image galleries, giving each title a more visual and immersive presentation than a plain text listing.

5-Star Rating System

An interactive star rating system lets the bookstore showcase how each title has been received, adding a layer of interactivity built with JavaScript.

Genre-Based Discovery

Books are organized by genre, helping readers browse toward titles that match their taste instead of scrolling through one long, undifferentiated list.

Key Features

Book Listings

Each book is displayed with cover art, description, and genre, giving readers enough detail to decide at a glance.

Interactive Ratings

The 5-star rating system gives the storefront a more dynamic, community-driven feel beyond static product pages.

Genre Discovery

Genre-based navigation helps visitors quickly find books in the categories they actually enjoy reading.

Responsive Layout

Built with responsive design principles, the bookstore adapts cleanly across desktop and mobile screens.
