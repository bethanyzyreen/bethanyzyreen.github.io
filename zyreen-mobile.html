About the Project

Built with Kodular

This Android application was developed using Kodular, a block-based visual programming platform. It demonstrates how mobile apps can be built without traditional code, using drag-and-drop components to create a functional and polished experience.

Bookstore Catalog

The app features a curated collection of books with cover images, descriptions, sizes, and Philippine Peso pricing. Users can browse titles across genres and add items to their cart through an intuitive product layout.

Green-Themed UI

The app uses a calming green color palette inspired by nature and growth, giving the bookstore a fresh, clean identity distinct from typical e-commerce apps. The Zyreen "Z" logo ties together the brand identity across all screens.

Multi-Screen Navigation

The app is structured with distinct screens — a home menu, a products catalog, an About Us section, and a Contact Us page — demonstrating how a complete mobile application flow can be created using visual block programming.

Key Features

Home Screen Navigation

A clean splash screen with the Zyreen brand logo and three main navigation buttons — About Us, Products, and Contact Us — giving users a clear and welcoming entry point to the app.

Add to Cart Functionality

Each book in the catalog includes an "Add to Cart" button, allowing users to select titles for purchase. Products display cover images, descriptions, physical dimensions, and pricing in Philippine Peso.

Visual Book Catalog

Books are displayed in a two-column grid layout with cover images prominently featured. Each card provides enough information for users to make an informed purchase without overwhelming the screen.

Consistent Brand Identity

The Zyreen "Z" logo with its crescent and wave motif appears across all screens, maintaining a cohesive green-toned visual identity that feels fresh and professional throughout the mobile experience.
